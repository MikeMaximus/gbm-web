Game Launcher F.A.Q

Why would I use Game Backup Monitor to my launch games?

  • Why not? Game Backup Monitor is always running in the system tray and usually has all the information required to launch any game you already monitor.
  • This makes GBM a lightweight and simple game launcher.
  • This feature is fully optional and completely invisible by default if you don't want to use it.

Why can't I launch games with Game Backup Monitor?

  • The game launcher feature is disabled by default, it must be enabled in the General section of File -> Settings.
  • This feature relies on the optional "Session Tracking" feature, it must be enabled as well.

How do I launch games with Game Backup Monitor?

  • You can launch any of the last five recently played games by right-clicking the system tray icon and choosing the game you'd like to play.
  • You can also launch any game from the Game Manager using the "Launch Options" button.

Why do I always get an error about missing information when trying to launch my games?

  • Unless you manually provided the information required to launch a game, it needs to be detected for at least one session before GBM has enough information to launch it.

Why can't I launch a game even after it's been detected once?

  • In rare cases, required information such as the "Game Path" can't be detected. The reason is usually displayed in the log. When this happens you can manually set the missing information in the Game Manager.

Why do some of my games crash when I launch them?

  • This is a problem with certain games purchased on services such as Steam or Epic Game Store.
  • Some games will simply refuse to work properly when launched directly from their executable and they need to launched with their respective store client. This can be DRM related or simply a bug.
  • GBM supports launching games using any store client, but the game must be manually configured to do so in the Game Manager. Please see the manual for more details.

I need to launch certain games as Administrator. Is this supported?

  • Yes. Any game flagged to "Run As Administrator" will request elevation as it normally does.
  • In addition, any game launched while GBM is running as Administrator will automatically launch as Administrator. However this is not recommended, and GBM will ask for confirmation before every launch.

I don't understand how Game Backup Monitor is launching my games. How does this work?

  • A priority system is used to decide how to launch games based on the information available.
    1. If a game has been manually configured to use a store client, it will be used to launch the game.
    2. If a game has been manually configured to use an alternate executable, it will be used to launch the game.
    3. If no information has been manually configured, the detected Game Path along with the configured Process and Parameter will be used to launch the game.
  • Please see the manual for more details and advanced information.
