What is Game Backup Monitor?

Game Backup Monitor is a Windows system tray application that automatically detects games as you play them. When you're finished playing, a compressed backup of the save data will be automatically created in a location you've specified, such as a cloud folder, network folder or external drive.

If you use Game Backup Monitor on multiple computers and share the same backup folder, you can effectively add cloud save support to any game or application. GBM can notify you of new save games to be restored when you move to each computer.

Game Backup Monitor was designed with classic DRM-free gaming in mind, but it can be used to monitor any application on your PC!

Game Backup Monitor is completely free and open source software.

Game Backup Monitor is currently only available in English.

Requirements

Supported Operating Systems:

Windows 10, Windows 8, Windows 7, Windows Vista SP1, Windows XP SP3

Recommended Minimum:

Pentium 1 GHz or higher with 512 MB RAM or more

Disk Space:

2 - 3 MB (This does not include the .NET 4 installation)


This software requires .NET 4 (Client Profile)

Windows 8 & 10 users do not need to install .NET 4.

Game Backup Monitor Features
  • Detects games as you play them and handles the backup automatically and immediately when you finish. No scheduling or input required.
  • Track the hours that you've played any monitored game, similar to Steam.
  • A growing list of automatically configured games.
  • Backup to a cloud folder and add cloud save support to any game.
  • Easily setup an unlimited number of games to monitor, each with their own specific settings.
  • GBM can detect and backup classic DOS games running in DOSBox or ScummVM.
  • Game settings are designed to be easily portable and shared.
