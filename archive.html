Information
  • This page contains game configuration lists that have been archived for use in old GBM versions.
  • Right-click and use "Save Link As..." to download the desired configuration list.
  • Left-click a configuration list to view and search for specific games. Archived configurations may show incorrect or missing data!
  • Use "Import -> File..." in the Game Manager to import from the XML file that was downloaded.
  • Archived lists are not updated, please use the latest version of GBM to access the most recent game configurations.
