Game Backup Monitor

Automatically backup your saved games with optional cloud support.

Cloud support requires any service that offers a folder synchronizing client, such as Dropbox.

New in 0.95

  • Bug Fixes
  • Fixed a regression that caused non-critical fields (Game Path, Company, Version, Icon, Enabled, Monitor Only) to be wiped or reset on sync.
  • The "Check for new backups" feature has been renamed and redesigned. Instead of an annoying pop-up, it now shows a simple notification in the main menu or tray menu.
  • Added the ability to trigger backups only after a certain session time has elapsed (Global Setting), this setting will be disabled by default.
  • You can now cancel out of the "Choose Game" window when GBM detects multiple games may be running.
  • Added the ability to filter by game information such as name, process and company.
  • Many UI improvements and fixes with filtering, such as the ability to filter by games with no tags. The "Tag" filter is now called "Custom" filter, due to the new options available.
  • Updated the "Add Game Wizard" to use the new include / exclude and tagging features.
  • Removed the special handling of DOSBox games for future proofing, DOSBox games will now be handled like all other games.
    • Due to this change, GBM will only detect DOS games that are using their own copy of DOSBox in a unique location. Classic games sold online are already packaged in this format.
    • This change WILL break all your current DOSBox game configurations and GBM can not reliably update them automatically. Sorry for the inconvience.
    • To repair a configuration, you'll need to delete the DOS executable part of the process field and make sure the case matches the DOSBox executable being used by the game. You can use the new "Custom" filter to easily find and update all your DOSBox games.
    • For example, Capitalism Plus using a process of "dosbox:CAPPLUS" and DOSBox.exe would become just "DOSBox".
    • All available official configurations for DOSBox games will be updated and ready for the v0.95 release.
  • Moved the import and export game list features from the Tools menu into the Game Manager.
  • Made some updates to the main app window. It looks a little less clunky!

  • Added a new, intuitive way of including and excluding items in a backup. A preview is below.

Notices
