Game Backup Monitor

Automatically backup your saved games with optional cloud support.

Cloud support requires any service that offers a folder synchronizing client, such as Dropbox.

Coming Soon

The next version (v0.95) is under development.

New in 0.95

  • Bug Fixes
  • You can now cancel out of the "Choose Game" window when GBM detects multiple games may be running.
  • Many minor UI improvements and fixes with tag filtering, such as the ability to filter by games with no tags.
  • Updated the "Add Game Wizard" to use the new include / exclude and tagging features .
  • Added the ability to trigger backups only after a certain session time has elapsed (Global Setting)
  • Removed the special handling of DOSBox games for future proofing, DOSBox games will now be handled like all other games.
    • Due to this change, GBM will only detect DOS games that are using their own copy of DOSBox in a unique location. Classic games sold online are already pre-packaged in this format.
    • This change will break your current DOSBox game configurations and GBM can not reliably update them automatically. Sorry for the inconvience.
    • To repair a configuration, you'll need to delete the DOS executable part of the process field and make sure the case matches the DOSBox executable being used by the game.
    • For example, Capitalism Plus using a process of "dosbox:CAPPLUS" and DOSBox.exe would become just "DOSBox".
    • All available official configurations for DOSBox games will be updated and ready for the v0.95 release.
  • Added a new, intuitive way of including and excluding items in a backup. A preview is below.

Notices
