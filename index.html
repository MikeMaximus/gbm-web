Game Backup Monitor

Automatically backup your saved games with optional cloud support.

Cloud support requires any service that offers a folder synchronizing client.

Changes in 1.01

  • (All) Add "Delete folder on restore" configuration option.
    • Some games are always changing the names of their save files, resulting in a mix of old and new saved games when syncing between computers.
    • This feature will recycle/trash the prior saved game folder before restoring your latest saves.
    • You can set this option on a per game basis.
    • This feature can only be used on configurations that backup an entire saved game folder and do not exclude any files.
  • (All) Add feature(s) to automatically handle new backups when they are detected.
    • Add option to automatically restore any new backup files when pre-requisites are met.
    • Add option to automatically mark new backups as restored when a game isn't currently installed.
    • These options are global and will be disabled by default. For more detailed information, see the manual after v1.01 is released.

Notices

  • Any changes listed are not final or complete.
