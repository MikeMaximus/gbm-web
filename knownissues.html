Known Issues & Solutions (v0.93)

Add Game Wizard

  • The Drag n' Drop shortcuts feature doesn't work when running GBM as Administrator.
    • Run GBM with UAC (User Access Control) off or run it with normal permissions.
  • The Drag n' Drop shortcut feature doesn't work when dragging shortcuts from the start menu on Windows 10.
    • Right-click the desired shortcut, choose "Open File Location", and drag the shortcut from the explorer window instead. You can also replace the terrible Windows 10 start menu with Classic Shell to solve this issue.

Game Detection

  • Very rarely GBM will stop detecting games, even when monitoring is enabled.
    • Quit and restart GBM. We've been unable to consistently replicate this problem, if you have any information on this issue please contact us.
  • GBM keeps asking for the executable path when a game is detected, even after it's already been set.
    • Open and close the Game Manager to force a reload after the game has ended. (Fixed in v0.94)
  • GBM may detect the incorrect game is running when games that share the same process name are running as administrator.
    • This is a bug with no acceptable workaround. (Fixed in v0.94)

Settings

  • The "Start with Windows" setting is on, but GBM isn't starting with Windows.
    • Open the Settings, untick "Start with Windows", then click Save. Open the Settings again, tick "Start with Windows" then click Save.

If your issue isn't listed here, please let us know about it.
