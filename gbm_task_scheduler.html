Using Task Scheduler to start Game Backup Monitor

This page will describe how to start Game Backup Monitor with Windows as Administrator with an optional delay.

These instructions are for Windows 10, they will differ slightly on other versions of Windows.

  1. Disable "Start with Windows" in Game Backup Monitor.
  2. Open Windows Settings, type "Task Scheduler" in the search and choose the "Schedule tasks" option when it appears.


  3. Right-click on the "Task Scheduler Library" folder and choose "New Folder..." from the menu.
  4. Enter "Game Backup Monitor" in the new folder window and click OK.
  5. Right-click on the "Game Backup Monitor" folder and choose "Create Basic Task..." to open the new task window.
  6. Enter "Startup" as the name of the task and any description you'd like. Click Next to move to the Trigger section.


  7. Choose "When I log on" and click Next to move to the Action section.


  8. Choose "Start a program" and click Next.


  9. Click the "Browse..." button and navigate to the folder where GBM is installed, choose the GBM.exe file. Click Next.


  10. The final page shows a summary. Tick the "Open the Properties dialog for this task when I click Finish" checkbox. Click Finish.


  11. The properties window will appear. Tick the "Run with highest privileges" checkbox to have GBM startup as Administrator. The next two steps are optional, you can click OK here and move to step #14 if you're finished.


  12. You may also want to set a short delay on the startup of GBM, this can improve the performance of Windows on a fresh boot. Click the "Triggers" tab, then the "Edit" button.


  13. Tick the "Delay task for:" checkbox and choose a desired time for the delay. Click OK to close the trigger window and OK again to close the properities.


  14. Close Task Scheduler.

Game Backup Monitor will now Start with Windows as Administrator with an optional delay.
