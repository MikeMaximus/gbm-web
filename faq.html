F.A.Q

Why should I use GBM over another backup solution?

  • GBM was developed to be a very simple but powerful "set it and forget it" backup solution. If other backup solutions seem too complicated, try this one!

    For detailed feature information and system requirements check the About section.

Why would I use this for Steam and Origin games when the client already has cloud backups?

  • Steam, Origin and other clients have cloud saving, but it's built strictly for convenience and gives the user very little control. GBM is also convenient, but gives you much more control over your backups if you want it.

    For detailed feature information and system requirements check the About section.

I don't like unfinished beta software, should I wait until the final version of GBM?

  • GBM may still be in a beta phase, but it is feature complete, reliable and safe to use. You can read about any known issues before downloading.

Is GBM safe to use and does it respect my privacy?

  • GBM does not perform any operation without request and consent until the user sets it up that way.

    GBM only connects to the internet to import game configuration data at the user's request. No information from your computer is ever uploaded to the internet by GBM.

    If you're really paranoid you can look at the source code available with each release.

How is GBM a cloud solution if it never uploads anything to the internet?

  • GBM is only a cloud solution if you want it to be. GBM needs to be paired with a cloud service that provides a folder sync client, such as OneDrive or Dropbox, to be a true cloud solution.

Why isn't GBM directly intergrated with a cloud service?

  • There was no point in doing so, GBM was designed with freedom of choice in mind.

    Do you want backup and share via a cloud? You can.
    Do you want to backup and share via a USB drive? You can.
    Do you want to backup and share via a network share? You can.

    You can also use GBM on a single computer if you like!

The Game Manager looks complicated, I thought this was supposed to be easy!?

  • The Game Manager might look a little scary, but the basic functions are pretty self-explanatory. There is a manual available that helps explain features in more detail.

    There's already a large amount of preconfigured games available from the official list and an "Add Game Wizard" is provided to help novice users setup their own games in GBM.

The official game list isn't updated very often and why aren't my games already configured?

  • GBM is currently a development team of 1. Installing, configuring and testing games with GBM requires a lot of time. Not to mention a copy of every game is required to do this properly.

Will GBM be avaliable for Linux or MacOS?

  • As of v0.96 GBM will run in Linux using Mono. MacOS has not been tested.
  • GBM also runs in Wine with limited usefulness. Check out the Wine FAQ for details.

Will GBM be ported to .NET Core?

  • .NET Core will not support Windows Forms, so the entire UI would have to be redone. I'm interested in doing this, but I can't commit to it at this time.

Will GBM be avaliable in other languages?

  • There are no plans to support other languages at this time. However, as of v0.95 GBM can be easily translated. Check out the Translation FAQ if you're interested.
