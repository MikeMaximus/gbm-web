How can I contribute to Game Backup Monitor?

Suggestions

If you have an idea that you think could improve Game Backup Monitor, don't be shy.

Send an e-mail or create a new issue.

Bug Reports

If you receive an error message or notice odd behavior, please take a few minutes and report the issue.

Game Configurations

If you'd like to help out with the official game configuration lists, please contact me.

Some knowledge of Git and GitHub are an asset if you'd like to help out.

Translations

Game Backup Monitor is only available in English, but anyone can help translate to other languages with minimal experience.

Please read the Translation FAQ if you're interested.

Financial Support

You can support the development of Game Backup Monitor via PayPal.

This helps maintain the hardware and services required to continue development of Game Backup Monitor.

Patreon with unique benefits may be available in the future.
